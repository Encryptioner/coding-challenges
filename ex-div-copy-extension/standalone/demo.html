How It Works (For Website Owners)

Step 1: Add One Script Tag

Add this to your website's HTML (before closing </body> tag):

<script src="textcopy.js"></script>
<script>TextCopy.init();</script>

Step 2: That's It!

Copy buttons automatically appear for your visitors. No configuration, no installation, no complexity.

Mobile Support

On mobile devices, copy buttons work perfectly too! Visitors can tap on content to see the copy button, or just tap the button directly.

The buttons are touch-friendly with large tap targets (44x44px minimum) for easy mobile use.

Customization (Optional)

You can customize the appearance and behavior:

TextCopy.init({
  buttonColor: '#FF5722',      // Change button color
  buttonText: 'Copy This',     // Change button text
  buttonPosition: 'top-left',  // Change position
  showOnHover: true,           // Show on hover only
  copyFormat: 'text',          // Copy as text or HTML
  showToast: true              // Show "Copied!" message
});

But most website owners don't need to customize anything - the defaults work great!
