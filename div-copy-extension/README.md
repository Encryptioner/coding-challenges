# DivCopy - Copy Any Element Chrome Extension

A Chrome extension that allows you to copy content from any div or element on any website with one click, similar to WordPress plugins like "Copy Anything to Clipboard" and "DivMagic".

## Features (In Development)

### Planned Features:
- ✓ Right-click context menu to copy element content
- ✓ CSS selector targeting for automatic copy buttons
- ✓ Copy as plain text, HTML, or formatted content
- ✓ Visual feedback when copying
- ✓ Customizable copy button appearance
- ✓ Support for nested elements
- ✓ Keyboard shortcuts

### Similar to WordPress Plugins:
- **Copy Anything to Clipboard** - One-click copy buttons
- **DivMagic** - Copy design elements from websites
- **Copy to Clipboard for WordPress** - Enhanced copy-paste functionality

## Installation

### From Source (Developer Mode)

1. Clone or download this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked"
5. Select the `div-copy-extension` folder

## Usage

1. **Right-click on any element** you want to copy
2. Select "Copy Element Content" from the context menu
3. Content is copied to your clipboard
4. Optional: Use the extension popup to configure settings

## Existing Alternatives

If you need this functionality now, consider these existing extensions:
- **DivMagic** - Copy elements as CSS/Tailwind components (paid)
- **Copy DOM Content** - Copy DOM elements (free)
- **SnipCSS** - Extract HTML/CSS for specific elements (free)
- **Copy Element Text** - Right-click to copy element text (free)
